
# Anikai CloudStream Provider

This is a CloudStream 3 extension for [https://anikai.to/](https://anikai.to/).

## Installation

1. Open CloudStream.
2. Go to Settings > Extensions > Add Repository.
3. Name: my-cloudstreamrepo
4. URL: https://raw.githubusercontent.com/hcgn2005-ai/my-cloudstreamrepo/master/repo.json
